1. Реализовать простейший parser/dumper json

   * `avarege_price`

   * `most_common_country`

   * `cheapest_wine`

   * `lowest_score`

	```

			"wine": {
				"Riesling": { ... },
				...
			},

			...
		}
	}

4. Оформить результаты из пункта `3` в виде красивого `markdown` файла

* статистика собрана верно
